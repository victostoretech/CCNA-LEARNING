# Network Devices
* Computer network: A digital telecommunication network which allows nodes to share resources.
* Client: A device that access a service made available by a server
* Server: A device that privides fuctions or services for clients
* Switch : A device that provides connectivity between hosts within the same network. Have many network interfaces compared to router(Usually 24+)
  Catalyst 9200,catalyst 3650(Enterprise grade switches)
* Router: A device that provides connectivity between LANs. Have fewer interfaces.
  ISR 900, ISR 1000, ISR 4000 etc
* Firewalls: network security device that controls or monitors traffic entering/exiting a network based on configured rules. Firewalls with more modern and advanced filtering capabilities are called Next-Generattion firewalls
  ASA 5500-X, firepower 2100
  Network Firewalls are hardwatre devices that filter traffic btw networks.
  Host-based Firewalls are software applications that filters traffic entering/exiting a host machine like PC.
